Supported file formats

  • gpx: this is the standard
  • kml: this it the standard developed by Google
  • kmz: a kmz file is simply a compressed kml file, hence it shares all its features
  • tcx: this is the standard introduced by Garmin; it supports
  • csv: this is a general data format; the program parses csv files from TomTom and Polar

Waypoints

Waypoints are automatically imported with tracks, or can be created by hand

Groups

Tracks can be manually assigned to groups of tracks. Some groups are also created automatically.

Main libraries used

Backend

  • Django 2.2 for the web application
  • gpxpy to read and export gpx files
  • tcx to read tcx files
  • fastkml and simplekml to read and export KML files
  • pandas to read csv files
  • geopy for geocoding

Frontend

  • Django Leaflet for maps
  • c3 for plots
  • DataTables for tables
  • leaflet-providers for the list of basemaps
  • d3
  • saveSvgAsPng
  • Leaflet.fullscreen
  • leaflet-groupedlayercontrol
  • jquery
  • jszip
