Supported file formats

  • gpx: this is the standard
  • kml: this it the standard developed by Google
  • kmz: a kmz file is simply a compressed kml file, hence it shares all its features
  • tcx: this is the standard introduced by Garmin; it supports
  • csv: this is a general data format; the program parses csv files from TomTom and Polar
Note that the format by TomTom, fit, is a binary file which cannot be freely read.

Main libraries used

  • gpxpy: to read and export gpx files (https://github.com/tkrajina/gpxpy)
  • tcx: to read tcx files (https://github.com/cwygoda/tcx)
  • geopy: for geocoding (https://github.com/geopy/geopy)
  • fastkml: to write KML files (https://github.com/cleder/fastkml)
